Network Stickies

Network Stickies (Assignment 4 of 6.170) is a simple web application that can be used to keep track of sticky notes. Users can:

  • Add, edit, and delete notes.
  • Organize notes by moving them with the mouse.
  • Access their own notes from any machine, as they are saved on the server
